Elements and Tags

1. What is an HTML tag?
- HTML tag is a container for an element in the web page

2. What are the three parts of an HTML element?
- Opening tag, content, closing tag

HTML Boilerplate

1. What is the purpose of the doctype declaration?
- To tell the browser what version of HTML it should use to render the document

2. What is the HTML element?
- The root element of the web page.

3. What is the purpose of the head element?
- Contains all the meta information, title tag, stylesheets, and scripts.

4. WHat is the purpose of the body element?
- Contains all the information that will be displayed to the web page

Working with Text
    How do you create a paragraph in HTML?
    - with the p tag
    How do you create a heading in HTML?
    - with h1 to h6 tag
    How many different levels of headings are there and what is the difference between them?
    - six, difference in emphasis and size
    What element should you use to make text bold and important?
    - strong tag
    What element should you use to make text italicized to add emphasis to it?
    - em tag
    What relationship does an element have with any nested elements within it?
    - an element becomes a parent when it contains a child element inside it
    What relationship do two elements have if they are at the same level of nesting?
    - they are sibling elements
    How do you create HTML comments?
    - with <!-- -->

Lists

    What HTML tag is used to create an unordered list?
    - ul + li tag
    What HTML tag is used to create an ordered list?
    - ol + li tag
    What HTML tag is used to create list items within both unordered and ordered lists?
    - li tag

links and images

    What element is used to create a link?
    - a element
    What is an attribute?
    - an attribute is something that gives an element extra definitions
    What attribute tells links where to go to?
    - href
    What is the difference between an absolute and relative link?
    - absolute link goes to pages of a different website, relative goes to pages of your own website
    Which element is used to display an image?
    - img
    What two attributes do images always need to have?
    - src and alt
    How do you access a parent directory in a filepath?
    - ../
    What are the four main image formats that you can use for images on the web?
    - jpg=best for photos, gif=best for animations, png=best for icons and diagrams, svg=best for responsive design
